# Social Image プロンプト (DALL-E)

GitHub リポジトリのソーシャルイメージ用の、温かみのあるスタイライズされたイラスト（1280x640ピクセル、2:1アスペクト比）。フレンドリーなヒューマノイドロボットと女性の人間の開発者が、同じデスクに並んで座り、1台のPCでキーボードを共有してペアプログラミングをしている。モニターには bash ターミナルが表示されており、暗い背景に緑色のテキストで「$ 」プロンプトと数行のコマンド、Linux ターミナル特有のカラフルな出力が映っている。ロボットはスリムだが親しみやすいデザインで、緑色に光る目を持つ。女性はカジュアルな服装で、リラックスした自信のある姿勢、楽しそうだが真剣な表情。両者は対等なパートナーとして画面を見つめている。デスクは居心地の良いモダンな雰囲気で、柔らかなアンビエントライティング。全体のカラーパレットはダークチャコール、ターミナルグリーン、暖色のアクセントを強調。微妙なグラデーション付き。テクノロジー系オープンソースプロジェクトに適している。
